starting steps in windows

mkdir first-electron-app # directory create
cd first-electron-app # directory change
npm init # initialize package.json
type nul > main.js # create main.js
type nul > index.html # create index.html
type nul > renderer.js # create renderer.js
type nul > styles.css # create styles.css
npm install --save-dev electron # install electron
code . # open visual studio code
npm start # start the electron app

the end
